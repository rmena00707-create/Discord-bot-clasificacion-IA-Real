# 🤖 Discord Bot - Clasificación de Imágenes IA vs Reales

Este proyecto es un **bot de Discord** capaz de analizar imágenes de gatos y determinar si fueron **generadas por Inteligencia Artificial** o si son **fotografías reales**.

## ✨ Características

- 🖼️ Analiza imágenes de gatos.
- 🤖 Detecta si la imagen fue creada con IA.
- 📊 Muestra el porcentaje de probabilidad de que la imagen sea IA o Real.
- ⚡ Respuesta rápida directamente en Discord.

## 🚀 Cómo usar

1. Escribe el siguiente comando en tu servidor de Discord:
   ```
   $check
   ```
2. Adjunta la imagen del gato que deseas analizar.
3. Espera unos segundos mientras el bot procesa la imagen.
4. El bot responderá con:
   - 📈 El porcentaje de probabilidad de que la imagen haya sido generada por IA.
   - ✅ O indicará que la imagen es **Real**.

## 📷 Ejemplo

```
Usuario:
$check
(Adjunta una imagen)

Bot:
🤖 Probabilidad de IA: 87.4%
📊 Resultado: Imagen generada por IA.
```

o

```
Usuario:
$check
(Adjunta una imagen)
```
